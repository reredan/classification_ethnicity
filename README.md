# Ethnicity Data Classification for Moldova

Analysis of ethnicity data from Moldova's Census (2004, 2014, 2024) and Ethnobarometer survey.

## Overview

This project simulates and analyzes multiple ethnic identity declarations using:
- Official census data from Moldova's National Bureau of Statistics
- Ethnobarometer survey data on ethnic self-identification
- Statistical modeling to estimate multiple identity patterns

## Data Sources

- **Census Data**: 2004, 2014, and 2024 censuses from [statistica.gov.md](https://statistica.gov.md)
- **Ethnobarometer**: Survey data from [HCNM Platform](https://files.hcnmplatform.org)

## Key Outputs

- Population estimates by ethnic group for 2024
- Analysis of multiple ethnic identity declarations
- Simulated data comparing single vs. multiple ethnicity reporting
